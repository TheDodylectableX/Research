# Dody's Research / Reverse Engineering Notes
This repository stores all of my RE notes as either binary templates for [SweetScape Software's 010 Editor](https://www.sweetscape.com/010editor/) program or standalone/program-specific scripts (Standalone Python scripts or Blender/Noesis etc.)
Stuff related to game file formats such as *(but not limited to)* models, textures and archives to name a few types. **All research is primarily made for PC versions of games unless it's a console specific game, Consult the binary templates for more details.**
If there's any credits with assistance in research it will be mentioned in the binary templates.

# Binary Templates - How to Use:
1. Download the repository's code as .ZIP, Extract the archive wherever you want but **don't change the folder structure to maintain compatibility with Common.bt as a dependency!**
2. To use the desired template make sure you have the right file that matches it's structure, Drag and drop the desired binary template into 010 Editor's window and press the play icon or F5 to execute it then select the file you want to run the template on, If done correctly you should be able to see template's results on screen and in the output section that it worked.

# Python Scripts - How to Use:
1. Check which one you want to use then download it.
2. Either drag and drop or use CMD to point out to it's directory then the target file.
